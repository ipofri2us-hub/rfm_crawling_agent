# 실패 사례: 70B급 대형 VLA 모델 VRAM OOM

- **시도한 모델**: 70B 파라미터 규모의 Vision-Language-Action(VLA) 정책 모델
- **환경**: 로컬 GPU 24GB VRAM
- **결과**: FP16으로도 추론 시 VRAM Out-Of-Memory 발생, 양자화(4bit) 적용 시에도
  배치 1 기준으로 KV 캐시 누적되며 장시간 구동 시 OOM
- **원인**: 파라미터 규모 대비 보유 GPU VRAM(24GB)이 부족함. 추론 시 필요한
  VRAM은 대략 "파라미터 수(B) x 2GB x 1.2" 이상으로, 70B 모델은 약 168GB 필요
- **결론**: 우리 하드웨어 스펙(VRAM 24GB)에서는 30B 이상 모델은 1차적으로
  실물 테스트 대상에서 제외. 동일/유사 규모의 신규 모델이 제안되면
  실물 평가 전에 반드시 양자화/경량화 가능성을 먼저 검토해야 함
